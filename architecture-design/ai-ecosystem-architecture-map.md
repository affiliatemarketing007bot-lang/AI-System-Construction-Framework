# AI Ecosystem Architecture Map

## Purpose

This document defines the top-level architecture of the AI system ecosystem and explains how the three repositories relate to one another.

The objective is to provide a single map that clarifies:

- how the system is organized
- what each repository is responsible for
- how methodology, platform architecture, and domain applications connect
- how future AI applications can be added without disrupting the core platform

This document is intended to reduce confusion as the ecosystem grows.

---

## Ecosystem Overview

1. **AI-System-Construction-Framework**
